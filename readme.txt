# IMPORTANT #
You need to have MongoDB local or remote database in order to run the application

How to Run Server:
1. cd server,
2. npm install,
3. create .env file
4. Add variable named MONGO_URI
5. Paste your MongoDB database URI as a value of MONGO_URI variable
6. npm run dev

How to Run Client:
1. cd client,
2. npm install,
3. npm run dev
